# How to run the example
```
workstation$ export TIDBCLOUD_PROJECT_ID=1111111111111111111
workstation$ export TIDBCLOUD_CLUSTER_NAME=APICluster
workstation$ export AWS_KEY_ID=xxxxxxx
workstation$ export AWS_SECRET_ACCESS_KEY=xxxxxxxxxxxxx
workstation$ export S3_URI=
workstation$ cd tidbcloud-sdk-go-v1/examples/DeleteCluster
workstation$ go run main.go
Succeeded in deleting the cluster
```
